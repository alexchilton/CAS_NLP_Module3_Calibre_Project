# Calibre Tools

A comprehensive set of tools for interacting with your Calibre library, including semantic search, duplicate detection, ISBN tools, and MCP integration for Claude.

## ‚ú® Features

- üîç **Semantic Search**: Find books using natural language queries powered by sentence transformers
- üìñ **Complete Metadata**: Get full book details including descriptions, publisher, ISBN, formats
- üöÄ **Batch Enrichment**: Automatically find and enrich books missing metadata from online sources
- üåê **Online Metadata Fetching**: Query Amazon, Goodreads, Google Books for rich metadata
- üîÑ **Duplicate Detection**: Find duplicate books by title, author, ISBN, or content similarity
- üìö **ISBN Tools**: Extract, validate, and search for ISBN-10 and ISBN-13
- üõ†Ô∏è **Calibre CLI Integration**: Python wrappers for all common Calibre operations
- ü§ñ **MCP Integration**: Use tools directly in Claude Desktop via Model Context Protocol
- ‚ö° **Performance Optimized**: Lazy loading, caching, and MPS support for Apple Silicon

## üîß Configuration

Configuration is handled via environment variables and `calibre_tools/config.py`:

### Environment Variables

```bash
# Calibre library path
export CALIBRE_LIBRARY_PATH="~/Calibre Library"

# Force cache refresh
export FORCE_REFRESH=1

# Cache expiry (days)
export CACHE_EXPIRY_DAYS=7

# Force CUDA (otherwise auto-detects MPS on Mac)
export USE_CUDA=1
```

### Device Detection

The system automatically detects the best available device:
1. **CUDA** - If `USE_CUDA=1` and CUDA available
2. **MPS** - If on Mac with Apple Silicon (with fallback)
3. **CPU** - Default fallback

### Cache Settings

Embeddings are cached in `~/.calibre_tools/`:
- `embeddings.pkl` - Book embeddings
- `metadata.json` - Book metadata

Cache is refreshed when:
- Files don't exist
- Files older than `CACHE_EXPIRY_DAYS`
- `FORCE_REFRESH=1` set

### Example Workflow: Batch Enrichment in Claude Desktop

Here's how to use the metadata enrichment tools to automatically enrich books missing metadata:

**1. Find books needing enrichment:**
```

‚Üí Uses calibre_find_books_needing_enrichment
‚Üí Returns list of books with IDs, titles, ISBNs
```

**2. Enrich a single book (automatic identifier detection):**
```
You: "Enrich book ID 1762"
‚Üí Uses calibre_enrich_book_metadata
‚Üí Auto-detects ISBN/ASIN from title or identifiers field
‚Üí Fetches metadata from Amazon/Goodreads/Google Books
```

**3. Apply updates selectively:**
```

‚Üí Uses calibre_apply_metadata_updates(1762, "Publisher,Series")
‚Üí Applies only the specified fields
‚Üí Returns confirmation of updates
```

**4. Batch process multiple books:**
```
You: "Batch enrich 20 books with ISBNs missing metadata"
‚Üí Uses calibre_batch_enrich_books(20)
‚Üí Processes up to 20 books automatically
‚Üí Returns summary: total processed, successful, failed
‚Üí Shows detailed results for each book
```

**Key Features:**
- ‚úÖ **Auto-detection** - Finds ASIN/ISBN in title or identifiers field
- ‚úÖ **Preview before applying** - See suggested changes before updating
- ‚úÖ **Selective updates** - Choose which fields to update
- ‚úÖ **Batch processing** - Enrich 10-50 books at once
- ‚úÖ **Focus on ISBNs** - More reliable than ASINs for metadata fetching

## üìñ API Documentation

### Semantic Search

```python
from calibre_tools.semantic_search import search, CalibreSemanticSearch

# Quick search
results = search("epic fantasy adventure", top_n=5)

# Advanced usage
searcher = CalibreSemanticSearch(
    library_path="~/Calibre Library",
    model_name="all-MiniLM-L6-v2",
    device="mps"  # or "cuda", "cpu"
)
results = searcher.search("query", top_n=10)
```

### Duplicate Finder

```python
from calibre_tools.duplicate_finder import (
    find_all_duplicates,
    find_exact_duplicates,
```

### Batch Enrichment

```python
from calibre_tools.batch_enrichment import (
    find_books_needing_enrichment,
    enrich_single_book,
    batch_enrich_books
)

# Find books with ISBNs that are missing descriptions

    limit=10,
    require_isbn=True,
    missing_fields=['comments', 'publisher']
)

for book in candidates:
    print(f"  {book['id']}: {book['title']} (ISBN: {book['isbn']})")

# Enrich a single book
result = enrich_single_book(book_id=1679)
if result['success']:
    print(f"Fetched metadata: {result['fetched_metadata']}")

    # Apply updates
    from calibre_tools.cli_wrapper import set_metadata
    set_metadata(
        book_id=1679,
        publisher=result['fetched_metadata']['Publisher'],
        comments=result['fetched_metadata']['Comments']
    )

# Batch process multiple books
results = batch_enrich_books(limit=10, find_candidates=True)
print(f"Processed: {results['total_processed']}")
print(f"Successful: {results['successful']}")
print(f"Failed: {results['failed']}")

for r in results['results']:
    if r['success']:
        print(f"‚úì Book {r['book_id']}: Enriched with {r['identifier_used']}")
    else:
        print(f"‚úó Book {r['book_id']}: {r['error']}")
```

## üêõ Troubleshooting

### Common Issues

**"calibredb not found"**
- Ensure Calibre is installed and in your PATH
- On Mac: `/Applications/calibre.app/Contents/MacOS/calibredb`

**MPS not working on Mac**
- Check PyTorch version: `python -c "import torch; print(torch.backends.mps.is_available())"`
- System falls back to CPU automatically

**Slow first search**
- First run downloads embedding model (~90MB)
- Subsequent searches use cached embeddings

**Import errors**
- Ensure all dependencies installed: `pip install -r requirements.txt`
